com_sendmail
============

This is an example component which demonstrates sending emails, both with and without using mail templates.

See [Example com_sendmail Component](https://manual.joomla.org/docs/general-concepts/mail/#example-com_sendmail-component) in the Joomla Manual
for a detailed description of how to use it.

Before using it, please ensure that your Joomla instance is configured for sending emails,
by pressing the Global Configuration / System tab / Send Test Mail button.
